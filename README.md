# Gym-Information-System

Diseño y desarrollo de un Sistema de Información basado en un análisis previo de requisitos y en un modelo de base de datos relacional.

El Sistema de Información consiste en una aplicación de escritorio bajo arquitectura cliente-servidor, programada en Java y utilizando la librería gráfica Swing. La aplicación puede conectarse a MariaDB, ofreciendo compatibilidad con el gestor de base de datos. La interacción con las bases de datos se gestiona mediante el ORM Hibernate.

Tecnologías empleadas:

- Entorno de desarrollo integrado (IDE), preferiblemente NetBeans
- Lenguaje de programación Java
- Librería gráfica Swing
- Sistemas Gestores de Bases de Datos: MariaDB
- Herramienta de mapeo objeto-relacional: Hibernate
